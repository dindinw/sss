my shadow sockets client service 

Intallation
-----------

1. Node.js
goto [http://nodejs.org/download/], verified in v0.10.26

linux

    wget http://nodejs.org/dist/v0.10.26/node-v0.10.26.tar.gz
    tar xzvf node-v0.10.26.tar.gz
    cd node-v0.10.26
    ./configure
    make
    make install

2. Shadow sockets

    npm install -g shadowsocks
